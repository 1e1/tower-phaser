Kapitel 1

Eine Kanone in Bewegung bringen

Wir legen das Fundament: wie ein Bild „lebendig wird“, wie eine Kanonenkugel einer Kurve folgt und wie wir das Ganze einpacken, um es zu teilen.

1Die Spielschleife: 60 Bilder pro Sekunde

Ein Spiel ist wie ein Zeichentrickfilm: eine Folge von Standbildern, sehr schnell gezeigt. Auf dem Bildschirm ergibt das die Illusion von Bewegung.

Etwa 60-mal pro Sekunde ruft der Browser eine Funktion auf, die wir update nennen. Bei jedem Aufruf bewegen wir die Objekte ein kleines Stück und zeichnen dann neu. Das ist die Spielschleife.

Warum dt? Nicht alle Computer laufen gleich schnell. Indem wir die Bewegungen mit der verstrichenen Zeit multiplizieren, läuft das Spiel überall gleich schnell. Clever!

2„Szenen“ wie im Theater

Wir teilen das Spiel in Szenen auf, ein bisschen wie die Zimmer eines Hauses: der Titelbildschirm, der Einstellungsbildschirm, das Match selbst. Wir wechseln von einer zur anderen.

  • 🏁 Boot — macht das Spiel bereit, der „Drücken zum Spielen“-Bildschirm.
  • ⚙️ Setup — wir wählen die Namen und die Anzahl der Runden.
  • 💥 Game — das Duell.
  • 🏆 Result — wer hat gewonnen?

Phaser kümmert sich von allein darum: Wir geben ihm die Liste und sagen „geh zur Game-Szene“.

3Die Flugbahn einer Kanonenkugel

Wenn die Kanone feuert, fliegt die Kugel geradeaus… dann zieht die Schwerkraft sie nach unten, und der Wind drückt sie zur Seite. Das Ergebnis ist eine schöne Kurve: eine Parabel.

4Docker: die Kiste zum Mitnehmen

Mein Spiel läuft auf meinem Computer… aber läuft es auch auf deinem? Oft braucht ein Programm bestimmte Werkzeuge (diese Version von jener Software). Fehlen sie, stürzt es ab.

Docker löst das: Wir packen das Spiel und all seine Werkzeuge in eine „Kiste“ (einen Container). Diese Kiste funktioniert überall gleich: auf deinem PC, auf einem Server, in der Cloud. Ein bisschen wie ein Schiffscontainer: egal welches Schiff, die Kiste ist immer dieselbe.

Merke dir. Etappe 1 = ein laufendes Spiel, in einer Docker-Kiste, bereit zum Teilen. Noch nicht hübsch: Das ist die Aufgabe von Kapitel 2!

Wie du baust…

…ein echtes Videospiel direkt im Browser

Verfolge die Entstehung von Tower Duel, einem Kanonenduell für zwei Spieler, Schritt für Schritt. Mit Diagrammen, echtem Code und kleinen Spielen, mit denen du herumprobieren kannst, um es zu verstehen.

?Was ist ein „Web-Spiel“?

Ein Web-Spiel ist ein Spiel, das in einem Browser läuft (Chrome, Firefox, Safari…), ohne dass du etwas installieren musst. Du klickst auf einen Link, und schon geht's los!

Damit das funktioniert, setzen wir drei Zutaten zusammen, die du vielleicht schon kennst:

  • 🧱 HTML — die Struktur der Seite (die „Bausteine“).
  • 🎨 CSS — das Aussehen: Farben, Größen, Layout.
  • ⚙️ JavaScript — das Gehirn: es sorgt dafür, dass sich Dinge bewegen.

Unser Spiel zeichnet alles in einen besonderen Bereich namens canvas (die „Leinwand“ eines digitalen Malers). Wir benutzen einen Werkzeugkasten, Phaser, der die kniffligen Teile übernimmt (Anzeigen, Animieren, auf die Tastatur hören), damit wir uns aufs Spiel konzentrieren können.

Merke dir. Der Browser weiß bereits, wie man Seiten anzeigt. Ein Web-Spiel ist „nur“ eine sehr lebendige Seite, auf der JavaScript das Bild dutzendmal pro Sekunde neu zeichnet.

★Das Spiel, das wir bauen

Zwei Türme, einer auf jeder Seite einer Landschaft. Jeder Spieler stellt den Winkel und die Stärke seiner Kanone ein und berücksichtigt dabei den Wind. Der Clou: Beide Spieler zielen gleichzeitig, dann werden beide Schüsse zusammen abgefeuert. Wer zuerst den gegnerischen Turm trifft, gewinnt die Runde!

Wir haben das Spiel in 4 Schritten gebaut (wir nennen sie „Etappen“). Jedes Kapitel erzählt einen Schritt, mit seiner eigenen Biom-Stimmung:

ETAPPE 1 · Wiese

Eine Kanone in Bewegung bringen

Die Spielschleife, die Flugbahn einer Kanonenkugel und wie wir das Spiel mit Docker „einpacken“.
ETAPPE 2 · Wüste

Es schön aussehen lassen

Kulissen (Biome), Partikel, Tiefe (Parallaxe) und Soundeffekte, die der Code erzeugt.
ETAPPE 3 · Tundra

Über mehrere Bildschirme spielen

Der Fernseher zeigt an, die Handys steuern. Ein Server schiedsrichtert das Match in Echtzeit.
ETAPPE 4 · Vulkan

Die Landschaft zerstören

Wie bei Worms: Jeder Schuss gräbt einen Krater. Wir sehen den Zaubertrick, mit dem man ein „Loch“ in ein Bild stanzt.

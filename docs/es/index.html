Cuaderno del creador
Cómo se construye…

…un videojuego de verdad dentro de un navegador

Sigue la creación de Tower Duel, un duelo de cañones para dos jugadores, paso a paso. Con diagramas, código real y pequeños juegos con los que puedes jugar para entenderlo.

?¿Qué es un “juego web”?

Un juego web es un juego que funciona en un navegador (Chrome, Firefox, Safari…) sin instalar nada. Haces clic en un enlace ¡y a jugar!

Para que eso funcione, juntamos tres ingredientes que quizá ya conoces:

  • 🧱 HTML — la estructura de la página (los “ladrillos”).
  • 🎨 CSS — el estilo: colores, tamaños, disposición.
  • ⚙️ JavaScript — el cerebro: es lo que hace que las cosas se muevan.

Nuestro juego dibuja todo dentro de una zona especial llamada canvas (el “lienzo” de un pintor digital). Usamos una caja de herramientas, Phaser, que se encarga de las partes complicadas (mostrar, animar, escuchar el teclado) para que podamos centrarnos en el juego.

Recuerda. El navegador ya sabe mostrar páginas. Un juego web es “solo” una página muy animada donde JavaScript redibuja la imagen decenas de veces por segundo.

★El juego que vamos a construir

Dos torres, una a cada lado de un paisaje. Cada jugador ajusta el ángulo y la potencia de su cañón, teniendo en cuenta el viento. El truco: ambos jugadores apuntan al mismo tiempo, y luego los dos disparos salen juntos. ¡El primero que dé a la torre contraria gana la ronda!

Construimos el juego en 4 pasos (los llamamos “fases”). Cada capítulo cuenta un paso, con su propio ambiente de bioma:

FASE 1 · Pradera

Hacer que un cañón se mueva

El bucle del juego, la trayectoria de una bala de cañón y cómo “empaquetamos” el juego con Docker.
FASE 2 · Desierto

Hacer que se vea bonito

Escenarios (biomas), partículas, profundidad (parallax) y efectos de sonido hechos con código.
FASE 3 · Tundra

Jugar en varias pantallas

La TV muestra, los móviles controlan. Un servidor arbitra la partida en tiempo real.
FASE 4 · Volcán

Destruir el paisaje

Como en Worms: cada disparo cava un cráter. Veremos el truco de magia para “hacer un agujero” en una imagen.

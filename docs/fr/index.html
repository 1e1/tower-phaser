Comment on construit…

…un vrai jeu vidéo dans un navigateur

Suis la fabrication de Tower Duel, un duel de canons à deux joueurs, étape par étape. Avec des schémas, du vrai code, et de petits jeux avec lesquels t'amuser pour comprendre.

?C'est quoi un « jeu web » ?

Un jeu web est un jeu qui tourne dans un navigateur (Chrome, Firefox, Safari…) sans rien installer. Tu cliques sur un lien, et ça joue !

Pour que ça marche, on assemble trois ingrédients que tu connais peut-être déjà :

  • 🧱 HTML — la structure de la page (les « briques »).
  • 🎨 CSS — la mise en forme : couleurs, tailles, disposition.
  • ⚙️ JavaScript — le cerveau : c'est lui qui fait bouger les choses.

Notre jeu dessine tout dans une zone spéciale appelée canvas (la « toile » d'un peintre numérique). On utilise une boîte à outils, Phaser, qui gère les parties compliquées (afficher, animer, écouter le clavier) pour qu'on puisse se concentrer sur le jeu.

À retenir. Le navigateur sait déjà afficher des pages. Un jeu web n'est « qu' »une page très vivante où le JavaScript redessine l'image des dizaines de fois par seconde.

★Le jeu qu'on construit

Deux tours, une de chaque côté d'un paysage. Chaque joueur règle l'angle et la puissance de son canon, en tenant compte du vent. Le piège : les deux joueurs visent en même temps, puis les deux tirs partent ensemble. Le premier à toucher la tour adverse gagne la manche !

On a construit le jeu en 4 étapes (qu'on appelle des « lots »). Chaque chapitre raconte une étape, avec son ambiance de biome :

LOT 1 · Prairie

Faire bouger un canon

La boucle de jeu, la trajectoire d'un boulet, et comment on « emballe » le jeu avec Docker.
LOT 2 · Désert

Le rendre beau

Les décors (biomes), les particules, la profondeur (parallaxe) et les effets sonores faits par le code.
LOT 3 · Toundra

Jouer sur plusieurs écrans

La TV affiche, les téléphones commandent. Un serveur arbitre la partie en temps réel.
LOT 4 · Volcan

Détruire le paysage

Comme dans Worms : chaque tir creuse un cratère. On verra le tour de magie pour « percer un trou » dans une image.
